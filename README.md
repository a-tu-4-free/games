## 📖 專案介紹 / Project Overview

這是一個基於 GitHub Pages 的網頁小遊戲合集網站，提供多款輕量級 HTML5 小遊戲，無需下載即可在瀏覽器中直接遊玩。

This is a GitHub Pages-based web game collection featuring lightweight HTML5 mini games. All games run directly in the browser without installation.

---

## 🚀 功能特色 / Features

- 🕹️ 多款小遊戲整合 / Multiple mini games in one platform  
- ⚡ 無需安裝即開即玩 / Instant play, no installation required  
- 📱 支援手機與電腦 / Fully responsive design  
- 🌍 GitHub Pages 免費部署 / Free hosting via GitHub Pages  
- 🧩 純前端架構 / Pure frontend (HTML + CSS + JS)  
- 🎮 輕量快速載入 / Lightweight and fast loading
