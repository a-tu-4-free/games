## 📌 專案簡介 / Overview

這是一個基於 **GitHub Pages** 建立的網頁小遊戲合集平台，收錄多款輕量級 HTML5 小遊戲，支援手機與電腦瀏覽器，無需下載即可直接遊玩。

This is a **GitHub Pages-based web game collection**, featuring lightweight HTML5 mini-games.  
All games run directly in the browser with no installation required.

---

## ✨ 特色 / Features

- 🕹️ 多款小遊戲整合 / Multiple mini-games in one place  
- ⚡ 即開即玩 / Instant play, no installation  
- 📱 手機 & 電腦支援 / Fully responsive design  
- 🌍 GitHub Pages 免費部署 / Free hosting via GitHub Pages  
- 🧩 純前端架構 / Pure frontend (HTML / CSS / JS)  
- 🎮 輕量快速 / Lightweight & fast loading  
- 🧠 易於擴充 / Easy to expand with new games
